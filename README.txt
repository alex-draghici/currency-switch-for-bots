=== WPML Currency Switch for Bots ===
Requires at least: 3.0.1
Tested up to: 3.4
Stable tag: 4.3

Switches currency for Google bots using wcml_geolocation_get_user_country hook from WooCommerce Multilingual & Multicurrency plugin.

== Description ==

Switches currency for Google bots using wcml_geolocation_get_user_country hook from WooCommerce Multilingual & Multicurrency plugin.

== Installation ==

1. Upload `currency-switch-for-bots.zip` to the `/wp-content/plugins/` directory
1. Activate the plugin through the 'Plugins' menu in WordPress
